# A Friend in the Kitchen

Hello! This is a shopping list and digital chalkboard for a dedicated tablet that lives in the kitchen.

## Setup

Configure variables in `.env`:
```
AIRTABLE_API_KEY=
AIRTABLE_BASE_ID=

DARKSKY_API_KEY=
LATITUDE=
LONGITUDE=
```
